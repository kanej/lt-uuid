UUID plugin for Light Table
---------------------------

Insert a new UUID at the cursor e.g. `9f4e9a51-a95f-4f62-afc7-60f1570d2255`.

Usage
-----

Only one command is currently available:

* UUID: Insert UUID at cursor. `:uuid.insert`
